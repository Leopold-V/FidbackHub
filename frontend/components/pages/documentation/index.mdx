# Quick start guide

Welcome on FidbackHub.\
To start with Fidbackhub, create a [new project](http://localhost:3000/project-creation).\
You can access your project dashboard and settings but in order to be able to vizualise any data coming from your application users, 
you'll need to set up the fidbackhub component.

## 1. Installation

You can install the Fidbackhub React component library as an NPM package

With Yarn:

```bash
yarn add fidbackhub
```

With npm:

```bash
npm install fidbackhub
```

## 2. Usage

Fidbackhub works as any other React component so It is extremely easy to start with.\
This is a small button which can be push to open a card with a form.\
The component has **an absolute position so be sure to put it in a relative container**.\
It will always be placed at the bottom-left of the container with a small margin.
The button has a default animation to be more "catchy" but can be disabled with a props (see next section).

**Example:**

<SyntaxHighlighter language="javascript" style={a11yDark}>

## 3. Project Token

You will need your project token for sending data. 

Copy your project token from your [project settings](http://localhost:3000/projects).
