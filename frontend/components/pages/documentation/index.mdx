# Quick start guide

Welcome on FidbackHub!\
To start with Fidbackhub, create a [new project](http://localhost:3000/project-creation).\
You can access your project dashboard and settings but in order to be able to vizualise any data coming from your application users, 
you'll need to set up the fidbackhub component.

## 1. Installation

You can install the Fidbackhub component by using the script tag:

<SyntaxHighlighter language="html" style={a11yDark}>
{`<script
      id="script_widget"
      type="text/javascript"
      src="https://cdn.jsdelivr.net/gh/Leopold-V/FidbackHub@0.1.0-alpha/widget/dist/fidbackhub-widget.js"
      data-key="YOUR_SECRET_KEY">
  </script>`}

</SyntaxHighlighter>

## 2. Project Token

You need your project token for sending data. 

Copy your project token in the project settings page.
This is the value you have to give above to the "data-key" attribute;
