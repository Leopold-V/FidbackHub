## Quick start guide

1. Installation

You can install the Fidbackhub React component library as an NPM package

<SyntaxHighlighter language="sh">
  {"npm install fidbackhub"}
</SyntaxHighlighter>

2. Project Token

You will need your project token for sending data. 

Copy your project token from your [project settings](http://localhost:3000/projects).

3. Usage

Use the component in your project

<SyntaxHighlighter language="react">
  {`import Fidbackhub from "Avisitor";

export const Fidbackhub = () => {
  return (
    <Fidbackhub token={YOUR_SECRET_KEY} />
  )
}`}
</SyntaxHighlighter>
